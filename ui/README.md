# 预演沙盘 - Streamlit 可视化 Demo

3 步完成奶茶店开店预演，零门槛体验「预演式决策」核心价值。

## 快速启动

```bash
# 在项目根目录
streamlit run ui/app.py
```

浏览器将自动打开 http://localhost:8501

## 使用流程

1. **左侧** 查看/修改参数（默认值为经典案例）
2. **点击「一键启动仿真」** 开始预演
3. **查看结果**：核心结论、时序图表、风险提示、完整报告

## 界面结构

| 区域 | 功能 |
|------|------|
| 左侧栏 | 参数配置（基础信息、成本、定价、市场） |
| 主区域顶部 | 核心结论看板、理想 vs 真实对比 |
| 主区域下方 | 标签页：时序图表、关键节点、风险提示、完整报告 |
| 底部 | 使用说明 |

## 依赖

- `streamlit` - Web 框架
- `pandas` - 图表数据
- `presim_core` - 仿真引擎、结果解析器

无需配置 API Key 即可运行。
