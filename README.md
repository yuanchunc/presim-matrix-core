## 🧋 经典爆款案例：25 万开奶茶店

> 对标 MiroFish 红楼梦演示 —— 用 AI 预演真实经营的连锁反应，呈现极致反差。

加盟商说：**月赚 2 万，半年回本**。  
真实预演告诉你：**约 90% 概率 6 个月现金流断裂闭店，累计亏损约 22 万**。

| 指标 | 加盟商宣传（理想） | 真实预演结果 |
|------|-------------------|--------------|
| 月均营收 | 6 万 | 约 1.2 万 → 0.5 万（逐月衰减） |
| 月均利润 | 2 万 | 持续亏损 |
| 回本周期 | 6 个月 | 未达盈亏平衡 |
| 6 个月后现金流 | 正（回本） | 断裂或濒临断裂 |

**为什么反差这么大？** 竞品促销分流、加盟成本、固定支出、淡旺季、消费者行为……真实世界的连锁反应，静态测算算不出来，**预演沙盘能**。

## 🛠 3 步快速上手

### 一、零门槛在线体验

本地运行 `streamlit run ui/app.py` 后访问 http://localhost:8501，无需配置，直接点击「一键启动仿真」体验奶茶店预演。部署至 Streamlit Cloud 可获得公开在线地址。

### 二、本地一键运行

```bash
# 1. 克隆项目
git clone https://github.com/yuanchunc/presim-matrix-core.git
cd presim-matrix-core

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动可视化 Demo
streamlit run ui/app.py
```

或运行脚本版（无 UI）：

```bash
python examples/milk_tea_shop/run_simulation.py
```

无需配置 API Key 即可跑通，结果自动导出 JSON / Markdown。

### 三、二次开发

修改 `examples/milk_tea_shop/config.yaml` 调整参数，或基于 `presim_core` 扩展自己的经营场景。详见 [扩展指南](docs/extension_guide.md)。

## 📁 项目架构

```
presim-matrix-core/
├── presim_core/       # 核心引擎（100% 开源）
│   ├── engine/        # 仿真调度（LangGraph）
│   ├── agents/        # 智能体模块
│   ├── llm/           # 大模型适配（OpenAI / Gemini / 通义千问）
│   ├── parser/        # 结果解析与风险识别
│   └── registry.py    # 扩展注册表（闭源模块入口）
├── examples/          # 示例（奶茶店 Demo 等）
├── ui/                # Streamlit 可视化（100% 开源）
└── docs/              # 文档
```

**开源 / 闭源边界**：核心引擎、UI、示例全部开源；深度归因、企业级能力通过 `Registry` 闭源扩展。
